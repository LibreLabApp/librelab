# LibreLab

> [!WARNING]
> **Early WIP (pre-MVP)**. Expect **messy/incomplete** code, embarrassing TODOs and unstable changes ([Initial Usable Release progress](https://github.com/LibreLabApp/librelab/issues/1)).
>
> The project is not in a usable state and is in active early development.
> It has been published early to establish development history for [Flathub requirements](https://docs.flathub.org/docs/for-app-authors/requirements#insufficient-development-history).

LibreLab is a self-hosted Laboratory Information System (LIS) for clinical diagnostic labs, integrating with medical analyzers, patient/result tracking, inventory, invoicing, and billing management, designed for lab operators.

Cross platform (Linux, Windows, macOS, Android, iOS) and cloud-deployable.
Free and open source software.
